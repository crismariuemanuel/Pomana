# FundraiseFrontend

Fundraising web app built with Angular 21 and Angular Material. It provides the UI for browsing campaigns, submitting donations, and managing user flows.

## Project structure
- `src/main.ts` / `src/app/*`: Angular bootstrap and feature modules/components.
- `src/styles.scss`: global styles.
- `public/`: static assets copied as-is.
- `angular.json`: Angular CLI workspace config.
- `package.json`: scripts and dependencies.


## Run locally (development)
```bash
npm install
npm start   
```

## Production build
```bash
npm run build 
```

## Unit tests
```bash
npm test
```

## Generate scaffolding (optional)
```bash
ng generate component component-name
```
